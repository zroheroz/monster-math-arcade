MONSTER MATH ARCADE v1.2 - DEPLOYMENT NOTES

BEST BETA DEPLOYMENT:
Use GitHub Pages, Netlify, Cloudflare Pages, or Google Cloud Storage static hosting.
The PWA install features require normal HTTPS hosting with index.html, manifest.webmanifest, sw.js, and icons in the same folder structure.

GOOGLE DRIVE ROLE:
Use Google Drive as your file vault:
- Store the ZIP package.
- Store parent backup JSON files.
- Share files between your computer and Lenovo tablet.
Google Drive is not the best true PWA host because service workers and manifest files must be served from a normal same-origin HTTPS site.

UPDATE FLOW:
1. Open Parent Dashboard.
2. Click Download Parent Backup.
3. Save the JSON backup to Google Drive > Kids Learning Apps > Backups.
4. Deploy or open the new version.
5. If progress is missing, open Parent Dashboard and Import Backup.

LENOVO INSTALL FLOW ONCE HOSTED:
1. Open the hosted HTTPS URL in Chrome.
2. Tap the three-dot Chrome menu.
3. Tap Install app or Add to Home screen.
4. Name it Monster Math.
5. Launch from the home screen.

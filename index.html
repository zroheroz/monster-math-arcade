Settings

Keep this simple for beta testing.

Version 1 beta rule: Do not push beyond 20 minutes. The win is daily reps, not marathon sessions.
Easier update flow: Parent Dashboard → Download Parent Backup → install/open the new app version → Parent Dashboard → Import Backup. This keeps progress safe when the tablet is fussy.
Install note: The best install experience requires the PWA package to be hosted on a normal HTTPS static host. Google Drive is excellent for storing backups and sharing files, but it is not reliable as true PWA hosting.
